## Installation

BitDebug is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BitDebug"
```

```swift
BitDebug.install(DummyViewController(), forciblyAlwaysDisplay: true)
BitDebug.show()
```
